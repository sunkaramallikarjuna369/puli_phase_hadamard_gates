üî¥ PAULI X GATE (Bit Flip)
The Pauli X gate performs a rotation of œÄ radians around the X-axis. It flips the qubit state: |0‚ü© ‚Üí |1‚ü© and |1‚ü© ‚Üí |0‚ü©.
Matrix Representation:
X = [0 1]
[1 0]
  • X|0‚ü© = |1‚ü©
  • X|1‚ü© = |0‚ü©
  • Eigenvalues: +1, -1
  • Classical equivalent: NOT gate
from qiskit import QuantumCircuit, QuantumRegister
from qiskit_aer import AerSimulator

# Create circuit
qc = QuantumCircuit(1, 1)

# Apply Pauli X gate
qc.x(0)

# Measure
qc.measure(0, 0)

# Simulate
simulator = AerSimulator()
job = simulator.run(qc, shots=1000)
result = job.result()
counts = result.get_counts(qc)
print(counts)
# Output: {'1': 1000}

üîµ PAULI Z GATE (Phase Flip)
The Pauli Z gate performs a rotation of œÄ radians around the Z-axis. It applies a phase flip: Z|0‚ü© = |0‚ü© and Z|1‚ü© = -|1‚ü©.
Matrix Representation:
Z = [1 0]
[0 -1]
  • Z|0‚ü© = |0‚ü©
  • Z|1‚ü© = -|1‚ü©
  • Introduces global phase -1
  • Only affects phase, not amplitude

‚ú® HADAMARD GATE
The Hadamard gate creates an equal superposition of |0‚ü© and |1‚ü©. It's crucial for quantum algorithms and performs a rotation of œÄ radians around the (X+Z)/‚àö2 axis.
Matrix Representation:
H = (1/‚àö2) [1 1]
       [1 -1]
  • H|0‚ü© = (|0‚ü© + |1‚ü©)/‚àö2
  • H|1‚ü© = (|0‚ü© - |1‚ü©)/‚àö2
  • Creates superposition
  • H¬≤ = I (applies twice = identity)
from qiskit import QuantumCircuit, QuantumRegister
from qiskit_aer import AerSimulator

# Create circuit
qc = QuantumCircuit(1, 1)

# Apply Hadamard gate
qc.h(0)

# Measure
qc.measure(0, 0)

# Simulate
simulator = AerSimulator()
job = simulator.run(qc, shots=1000)
result = job.result()
counts = result.get_counts(qc)
print(counts)
# Output: ~50% |0>, ~50% |1>
